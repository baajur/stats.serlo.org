# Data Analysis and KPI Monitoring

** This project is in prototyping phase! **

## Getting Started

1. Install Fluentd
2. `fluentd -c conf/fluent.conf`
3. `python3 mfnf_edits.py`

The fluentd log should now show some entries on stdout.
